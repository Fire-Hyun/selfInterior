# Phase 2 ExpertLead 테스트 케이스

## 전문가 조회

- 전문가 카테고리 목록은 active 항목만 조회된다.
- 전문가 목록은 `ACTIVE` 상태만 노출된다.
- 카테고리, 지역, 예산, 부분 시공 가능 여부 필터가 적용된다.

## 프로젝트 추천

- 최근 질문 risk level이 높으면 질문 문맥 기반 카테고리가 우선 추천된다.
- 질문 기반 추천이 없으면 현재 공정 단계 기반 카테고리가 사용된다.
- 프로젝트 홈 추천 전문가 카드가 실제 추천 결과를 반영한다.

## 리드 생성

- 프로젝트에서 전문가 리드를 생성할 수 있다.
- `attachment_payload`에 프로젝트 요약, 주소 요약, 최근 질문 요약이 저장된다.
- 리드 생성 시 `expert_lead_events`에 생성 이벤트가 함께 기록된다.
