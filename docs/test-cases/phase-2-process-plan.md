# Phase 2 ProcessPlan 테스트 케이스

## 공정 플랜 생성

- 선택된 도면 후보가 있으면 프로젝트별 공정 플랜이 생성된다.
- 프로젝트 유형별로 단계 조합이 달라진다.
- 선택된 도면 후보가 없으면 생성이 거부된다.

## 공정 플랜 조회

- 단계 목록, 현재 단계, 진행 상태를 조회할 수 있다.
- 단계 상세 조회 시 목적, 결정 포인트, 체크리스트가 반환된다.

## 체크리스트 토글

- 태스크 완료 상태를 변경할 수 있다.
- 모든 태스크가 완료되면 단계 상태가 갱신된다.
- 현재 단계가 바뀌면 프로젝트 홈과 `projects.current_process_step`에 반영된다.
