# Phase 3 Style 테스트 케이스

## 스타일 생성

- 프로젝트에서 스타일 이미지를 생성할 수 있다.
- 생성 결과에 prompt text, model name, metadata, liked=false가 저장된다.
- 공간별 이미지 카드가 목록에 노출된다.

## 스타일 선택

- 이미지 좋아요를 누르면 liked 상태가 갱신된다.
- 같은 공간에서 기존 선택이 있으면 해제되고 새 선택이 저장된다.

## 스타일 페이지

- preset 선택, 공간 선택, 추가 프롬프트 입력 후 결과를 확인할 수 있다.
- 카드에 난이도, 예산 영향, 필요한 공정 힌트가 함께 표시된다.
