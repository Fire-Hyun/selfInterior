# Phase 2 VisualQuestion 테스트 케이스

## 질문 등록

- multipart 요청으로 질문과 이미지를 등록할 수 있다.
- 이미지 메타데이터와 storage path가 저장된다.
- 답변이 동기 생성되고 질문 상태가 완료로 저장된다.

## 질문 조회

- 프로젝트별 질문 목록을 최신순으로 조회할 수 있다.
- 질문 상세 조회 시 답변과 이미지 목록이 함께 반환된다.
- risk level과 expert_required 값이 응답에 포함된다.

## 프로젝트 홈 연동

- 최근 질문이 있으면 프로젝트 홈 카드가 placeholder 대신 실제 질문 요약을 보여준다.
