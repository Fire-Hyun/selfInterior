
# Codex 요청용 스크립트 모음
작성 기준일: 2026-03-26

이 파일은 Codex에 순차적으로 붙여 넣기 위한 프롬프트 모음이다.
기본 가정:
- monorepo
- Next.js App Router + TypeScript + Tailwind
- Spring Boot + PostgreSQL + Redis + S3
- 모듈형 모놀리스
- 주소-first, 도면-fallback 설계

==================================================
[MASTER PROMPT]
==================================================

너는 시니어 풀스택 아키텍트이자 구현자다.
내가 만들려는 서비스는 “반셀프 인테리어 웹 서비스”다.

핵심 기능:
1. 사용자가 아파트명/주소/동·호를 입력하면 집 정보를 자동 식별
2. 가능한 범위에서 도면을 자동으로 찾거나, 못 찾으면 근사 평면을 생성
3. 그 집 기준으로 스타일 예시 이미지 생성
4. 공정 단계별 체크리스트/가이드/주의사항 제공
5. 현장 사진 질문답변 제공
6. 필요한 시점에 공정별 전문가 연결

중요한 비즈니스 원칙:
- 웹 우선, 앱은 나중
- 주소-first, 도면 업로드는 fallback
- 도면이 없더라도 제품 흐름이 끊기면 안 됨
- provenance, license_status, confidence_score를 반드시 저장
- 매출은 전문가 리드와 유료 리포트에서 발생
- 고위험 답변은 확정 표현 금지

기술 원칙:
- 외부 API는 반드시 adapter interface 뒤에 숨길 것
- mock provider로 먼저 완성하고 실제 연동은 교체 가능하게 설계
- API 실패를 정상 플로우로 다룰 것
- 테스트 가능하고 문서화된 코드 작성
- 공통 응답 포맷, 에러 코드, requestId, 로깅 구현
- DB migration 포함
- README 충실하게 작성

작업 방식:
- 내가 단계별로 기능 구현을 지시할 것이다.
- 각 단계마다:
  1. 변경 계획 요약
  2. 파일 생성/수정
  3. 코드 구현
  4. 테스트 추가
  5. 실행 방법/검증 방법 정리
를 반드시 포함하라.
